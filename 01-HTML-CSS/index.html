안녕하세요, 프론트엔드 신입 개발자 최소연입니다.

사용자가 직관적이고 편하게 웹 서비스를 이용할 수 있도록 UI/UX에 집중하여 개발하는 것을 좋아합니다.
어떤 구성과 사용 흐름이 사용자에게 좋을 지를 항상 고민합니다.
또한, 현재는 코드를 작성할 때 성능을 고민하여 사용자에게 보다 좋은 경험을 제공할 수 있는 개발자가 되려고 노력하고 있습니다.

주도적으로 행사를 기획하고 운영하는 것을 즐기며, 다양한 동아리와 교육 프로그램에서 MT, 파티, 대회, 이벤트, 축제 부스 등을 운영한 경험이 있습니다.
이런 행사 운영을 통해 사람들과 소통하고 감정을 교류하는 것을 좋아합니다.
